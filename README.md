# Keti Transports

Site vitrine de **Keti Transports**, entreprise spécialisée dans le transport, l’évacuation et la livraison de matériaux de chantier en Suisse romande.

## Présentation

Ce site met en avant les prestations de Keti Transports pour les professionnels et particuliers qui ont besoin d’une solution fiable pour leurs chantiers : évacuation, transport, fourniture de matériaux et livraison de béton.

## Services présentés

- Évacuation de déblais et de matériaux de démolition
- Transport de déchets de chantier
- Mise à disposition de camions basculants avec chauffeur
- Livraison de gravier, terre végétale, béton et autres matériaux
- Intervention en Suisse romande
- Informations de contact et demande via formulaire

## Points forts du site

- Design responsive pour mobile, tablette et ordinateur
- Page unique claire et rapide à consulter
- Optimisation SEO de base
- Balises Open Graph pour le partage sur les réseaux sociaux
- Données structurées JSON-LD pour améliorer la compréhension par les moteurs de recherche
- Fichiers `robots.txt` et `sitemap.xml` inclus

## Technologies utilisées

- HTML5
- CSS intégré dans `index.html`
- JavaScript vanilla
- Assets image dans le dossier `img/`

## Lancement local

Le site peut être ouvert directement dans un navigateur :

```text
index.html
```

Il peut aussi être servi avec n’importe quel serveur statique, par exemple via l’extension Live Server de VS Code.

## Publication

Le projet peut être publié sur GitHub Pages ou sur un hébergement web statique classique.

Fichiers à publier :

- `index.html`
- `robots.txt`
- `sitemap.xml`
- `img/`

Éléments volontairement exclus du dépôt :

- `.idea/` : configuration locale de l'IDE ;
